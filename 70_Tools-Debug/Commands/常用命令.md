# 常用命令

常用 adb / dumpsys 命令统一放在这里。命令只沉淀取证方法，业务判断回到对应流程和案例。

## adb命令

### 基础

```powershell
adb devices
adb root
adb remount
adb shell getprop
adb shell date
```

### logcat

```powershell
adb logcat -c
adb logcat -b main,system,radio -v threadtime > ap_radio.log
adb logcat -b radio -v threadtime > radio.log
adb logcat -v threadtime | findstr /i "RILJ ServiceStateTracker Ims Uicc CarrierConfig"
```

### bugreport

```powershell
adb bugreport bugreport.zip
```

### 常用状态

```powershell
adb shell dumpsys telephony.registry
adb shell dumpsys phone
adb shell dumpsys ims
adb shell dumpsys carrier_config
adb shell dumpsys connectivity
adb shell dumpsys wifi
```

### SIM/网络相关属性

```powershell
adb shell getprop | findstr /i "gsm sim ril radio ims persist.vendor"
adb shell service call phone 1
```

## dumpsys命令

### Telephony

```powershell
adb shell dumpsys phone
adb shell dumpsys telephony.registry
adb shell dumpsys isub
adb shell dumpsys carrier_config
```

### IMS

```powershell
adb shell dumpsys ims
adb shell dumpsys telecom
```

### Connectivity / Wi-Fi

```powershell
adb shell dumpsys connectivity
adb shell dumpsys wifi
adb shell dumpsys netpolicy
```

### 分析建议

- `dumpsys` 是状态快照，不是完整过程。
- 对注册、IMS、call failure，仍然需要 failure window 的 AP log 和 modem log。
- 保存 `dumpsys` 时记录采集时间，方便和 log 对齐。
